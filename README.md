# Article Visual Explainer

An AI-agent skill for turning a complete article or chapter into a source-faithful, mobile-readable, single-file HTML visual explainer.

The skill supports URL, WeChat, Markdown, HTML, PDF, OCR text, and scanned PDF sources. It includes source-routing guidance, an evidence-ledger workflow, visual-system rules, an HTML shell, and validation scripts.

## Contents

- `SKILL.md` — main skill instructions
- `references/` — source acquisition, analysis, visual, HTML, and layout contracts
- `assets/explainer-shell.html` — starter HTML shell
- `scripts/` — HTML and layout validators

## Scope

Use this skill when the desired output is a validated, shareable HTML explainer. For a simple summary or bullet list, a lighter workflow is usually more appropriate.
